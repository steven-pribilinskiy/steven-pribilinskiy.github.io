# PRODUCT.md

## Register

brand — a personal portfolio; design IS the product.

## Product

Steven Pribilinskiy's personal profile site, served at https://steven-pribilinskiy.github.io. Single page: hero, stats ledger, about, experience timeline, expertise index, contact. No projects showcase, no blog, no Aylith branding.

## Users

Recruiters, engineering leaders, and OSS collaborators deciding within ~30 seconds whether this person is worth talking to.

## Brand personality

"Editorial Engineering" — a typographically serious magazine page whose developer identity is carried by monospace metadata and a single terminal artifact (the hero session window). Confident, literate, calm. Warmth comes from typography and the paper/ink palette, never from decoration.

## Anti-references

Glassmorphism, neon gradients, purple-gradient heroes, emoji bullets, card grids with hover lift, pill chips, typed-name animations, skill bars, count-up counters, full-terminal cosplay (green-on-black, scanlines), "Hi, I'm X 👋" openers.

## Design principles

- One accent (burnt orange / ember) + one micro-accent (signal green, status dot and prompt char only).
- Exactly one shadowed element on the page (the session window).
- The page is square: radius only on the session window (10px) and inline code (4px).
- Numbered sections are the deliberate editorial motif of this one page — kept intentionally.
- Zero layout shift on hover/theme/interaction. Light and dark are both first-class (warm h≈95 family in both).
- Near-zero JS: theme toggle and the years-since counter are the only scripts.
